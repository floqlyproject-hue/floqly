# Sandbox 1: Plasma

## Описание
Плазменный эффект внутри круга с переливающимися формами. Основан на CSS анимации с SVG масками.

## Источник
Uiverse.io by andrew-manzyk

## Особенности
- Плавная анимация (16s цикл)
- Остановка при клике
- Иконка закрытия при раскрытии
- Иконки соцсетей с градиентами
