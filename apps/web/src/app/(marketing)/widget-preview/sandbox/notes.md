# Sandbox - Песочница для тестирования

## Назначение

Страница для быстрого тестирования новых дизайнов виджетов перед созданием полноценной страницы.

## Как использовать

1. Положи референс в `references/` или код в `source/`
2. Напиши мне что положил
3. Я адаптирую код и вставлю в `page.tsx`
4. Смотришь результат на http://localhost:3000/widget-preview/sandbox
5. Даёшь фидбек, итерируем
6. Когда готово — переносим на постоянную страницу дизайна

## Что есть на странице

- Переключатель темы (светлый/тёмный фон)
- Кнопка "Назад"
- Пустое пространство для виджета
