# Design 01 — Simple Widget (Plasma)

## Описание стиля
Плазменный эффект внутри круга с переливающимися формами. Основан на CSS анимации с SVG масками. Градиенты от оранжевого к красному с hue-rotate анимацией для создания эффекта "живой" плазмы.

## Референсы
Uiverse.io by andrew-manzyk
https://uiverse.io (оригинальный источник)

## Исходный код
См. `/source/1.md` — оригинальный HTML/CSS код от andrew-manzyk

## Особенности реализации
- Плавная анимация плазмы (16s цикл)
- Остановка анимации при клике (класс `.paused`)
- Иконка закрытия (X) появляется при раскрытии
- Иконки соцсетей (WhatsApp, Telegram, VK) с градиентами
- Staggered анимация появления иконок (60ms задержка между ними)
- Glossy эффект на кнопках соцсетей

## Заметки
- Виджет использует custom CSS (`plasma.css`) вместо Tailwind для анимаций
- SVG маска создаёт эффект "живых" форм внутри круга
- Цвета можно легко менять через CSS переменные (`--color-one`, `--color-two` и т.д.)
- Время анимации: `--time-animation: 16s` (можно ускорить/замедлить)
- Размер виджета: `--size: 0.7` (масштабирование от оригинальных 100px)
