# /widget-check — Проверка виджета

Проверь состояние embed-скрипта виджета.

## Что проверить

1. **Bundle size:**
   ```bash
   cd apps/widget && pnpm build
   # Проверить размер dist/widget.js (gzipped < 50kb)
   ```

2. **Shadow DOM изоляция:**
   - Все стили внутри Shadow Root
   - Нет глобальных CSS

3. **Совместимость:**
   - Работает в современных браузерах
   - Не конфликтует с React/Vue/jQuery
   - Работает внутри iframe

4. **Производительность:**
   - Загрузка < 200ms
   - Не блокирует рендеринг страницы
   - defer/async загрузка

5. **Безопасность:**
   - CORS настроен правильно
   - Нет XSS уязвимостей
   - Нейтральные имена классов (не блокируется AdBlock)

## Отчёт

Выведи результаты проверки в формате:
- Bundle size: X kb (OK/WARN/FAIL)
- Shadow DOM: OK/FAIL
- Совместимость: OK/ISSUES
- Производительность: OK/WARN
